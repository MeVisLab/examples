Learn to use MeVisLab

After receiving a lot of useful feedback regarding the MeVisLab documentation, we decided to provide you with simple yet well-thought-out tutorials and common examples to allow you to gain profiency in the usage of MeVisLab quickly and easily.

As oftenly requested by users and customers, we split the existing documentation into separate chapters of functional categories to furtherly increase the usability and simplify the access to explicitly needed information.

Find some interesting links below if you feel like you are ready for a quick start:

  • Tutorial Overview
  • Getting Started Manual
  • Example Networks
  • MeVisLab forum

We are still working on the contents of this page and all related resources, so please be patient and continue to support us by giving feedback on the existing parts. If you want to contribute any ideas, example networks or topics for additional tutorials, you may do so by adding them to our repository on Github. Please make sure to provide us with a Markdown file containing a description of your suggestion (and/or a screenshot of your example network) when filing a pull request.

Discern how many of our tutorials are on YouTube as well.

Lastly: We implemented a basic search functionality for you to search for terms within our tutorial page.
